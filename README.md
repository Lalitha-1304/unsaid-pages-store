# Unsaid Pages Store

Unsaid Pages Store is a responsive single-page e-commerce application
that allows users to browse and purchase diaries through a clean and
modern user interface.

## Overview
This project is a frontend-only e-commerce website developed using
React.js and TypeScript. It is built with Vite for fast development and
optimized performance. The application uses React Router for smooth
page navigation and Context API for global state management.

The focus of this project is to deliver a seamless shopping experience
with a responsive design and intuitive user interface.

## Technologies Used
- React.js
- TypeScript
- Vite
- HTML
- CSS
- React Router
- Context API
- Lucide Icons

## Features
- Responsive single-page application (SPA)
- Product listing for diaries
- Smooth navigation using React Router
- Global state management with Context API
- Modern UI with Lucide icons
- Fast build and development using Vite
- Deployed as a static frontend application
